# Naive-ui Dashboard Template

This template is taken from [Mx Space Admin v2](https://github.com/mx-space/admin-vue3) and styled the same way.

# Usage

Just use it as a regular template, you can style it in 'configs.json'.

```
pnpm i
pnpm run dev
```
